# 로맨스 장르 작품 랭킹 리스트

```shell
// 아래 레포지토리에서의 서버 구동이 필요합니다.
// https://github.com/zue-dev/lezhin-server

$ yarn
$ yarn start
```

## 사용 라이브러리

### Redux-toolkit

보일러플레이트를 줄이고 아키텍쳐상의 관심사분리를 위해 사용했습니다.

### axios

보안적으로 fetch보다 낫고 간결하게 쓸 수 있어 사용했습니다.

### styled-components

컴포넌트 재사용시 스타일을 한 파일 안에 둘 수 있는 것이 장점이라고 생각됩니다.
